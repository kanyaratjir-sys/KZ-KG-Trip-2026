🔥 ตั้งค่า Firebase Sync

1. ไปที่ console.firebase.google.com → Add project
2. Build → Realtime Database → Create database → เลือก Singapore → Test mode
3. กลับไป Project Settings (⚙️) → General → Your apps → กด </> (Web)
4. ตั้งชื่อ app → Register → จะเห็น firebaseConfig → Copy ทั้งหมด
5. วาง Config ด้านล่าง แล้วกด "เชื่อมต่อ"

⚠️ Test mode ใช้ได้ 30 วัน หลังจากนั้นต้องแก้ Rules ที่ Firebase Console
